                  ====Users and groups commands====
-------------------------------------------------------------------
1. adduser username - create a user
2. adduser username groupname - add user to a specific group
3. cat etc/passwd - to see users
4. users - to see users
5. groups - to see groups
6. getent group - to see all groups
7. id username - tells all details about users
8. who - tells who loggedin 
9. usermod --lock username - to lock a user
10.usermod --unlock username - to unlock a user
11. groupadd groupname - create a new group
12. sudo usermod -aG groupname username - add a user to a group
13. sudo gpasswd -d username groupname - remove a user from a group
14. sudo groupmod -n newname oldname - change a group name
15. sudo groupmod -g NEW_GID groupname - change a group id
16. sudo usermod -g newgroup username - change user's primary group
17. sudo usermod -aG groupname username - add user to secondary    group
18. sudo groupmod -n newname oldname -change a group name
19. sudo deluser username - to delete a user
20. sudo deluser --remove-home username - dlt a user and also its home dir
21. sudo deluser username groupname - dlt user from a specific group
22. sudo delgroup groupname - to dlt a specific group
23. sudo usermod -g primarygroup -G group1,group2 username - set both primary and secondary group at one time
24. sudo usermod -aG group1,group2,group3 username - add multiple secondary groups
25. sudo usermod -G group1,group2 username - replaceall secondary group
26. sudo usermod -g groupname username - add primary group
27. psswd username - to modify psswd

                     ===File Permissions==
Permission types:-
r=read(4)
w=write(2)
x=execute(1)

Permission Groups:-
u=users
g=group
o=owner

644 = create a file with user
775 = create a dir with user(when gp and user is same)

755 = create a dir with root
644 = create a file with root


1. ls -l = to view permissions
2. chmod = to change the permissions
3. chmod u=rwx,g=rx,o=r filename = to change the permission
4. sudo chown username filename = to change the owner of the file
5. sudo chown :groupname file = to change group of the file
6. sudo chown user:group file = change both group and user

=>screen:-
screen is a terminal multiplexer that allows you to:

✔ Run processes in the background
✔ Keep sessions alive even after logout
✔ Reconnect to running sessions
✔ Run multiple terminals inside one SSH connection

1. screen = unname session.
2. screen -S mysession = name session
3. screen -ls
4. screen -r mysession = reattach session
5. ctrl + A , d = detach session
6. screen -S myapp -dm node app.js = run command direct on screen
